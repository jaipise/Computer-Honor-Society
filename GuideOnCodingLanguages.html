Best Programming Languages to Learn 2020-2021

by William Bae

Python3

Python is one of the best languages to learn for beginners. A dynamically typed language (you don't have to declare your variables), Python is flexible and easy to use. A general-purpose language that can do just about anything, but generally slower and less efficiently than other languages. Many a developer has picked this language up, making Python one of the most popular programming languages in the world, especially in fields such as artificial intelligence, machine learning, and data science.


  • Renown for being an easy to pick up
  • Is an Object Oriented Programming language
  • Large user base and resource availability
  • Very flexible in application
  • Cross-platform support
  • Relatively concise code

Cons:
  • Slower than many other languages
  • Weak with memory intensive tasks
  • Worse at mobile develepmont
  • Dynamic Typing may lead to issues at run-time

Java

Another great language for beginners. Based off of C, Java is slower than its predecessor, but also simpler to use. More prominently, although it is based off of C, Java is platform independent, allowing Java to become another one of the most popular languages to use in 2020, great for any application development.


  • Another fairly easy language to learn
  • Object Oriented Programming
  • Large user base and resource availability
  • Flexible in application
  • Cross-platform support

Cons:
  • Slower than natively compiled languages(eg. C/C++)
  • More difficult than languages like Python

C/C++

One of the oldest languages, C has stood the test of time and is the fundamental language many others are based off of. Powerful and efficient, C and C++ differ in that the former is not an Object Oriented Programming language, but as a general rule, any C program is compatible as a C++ program as well. While the two languages share a general difficulty to pick up for beginners, those that master the language will be rewarded through the language's inherent power and control over the use of the computer's resources.


  • Fast – As an interpreted language, compilation time is reduced
  • Easy to pick up and use
  • Object Oriented Programming
  • Very Popular + Widespread Community Support

Cons:
  • Hard to debug
  • Not Object Oriented Programming(C)
  • Harder to pick up

JavaScript

Although JavaScript may sound similar to Java, don't confuse the two – they are completely unrelated. Like Python, JavaScript is dynamically typed. Though now capable of application software development, JavaScript is most often seen in interactive web development, and is the most popular language on Github to date. Many reputable websites like Netflix and Paypal utilize JavaScript to create dynamic, interactive webpages. It's a great, fairly simple to learn language to use alongside other languages(like HTML) and to dive into web development.


  • Fairly Easy to use
  • Object Oriented Programming
  • Large user base and resource availability
  • Great at large web development projects
  • Works well with other languages/li>

Cons:
  • Less common outside of web development
  • Relatively more difficult to debug due to interactions with browsers

Swift

Introduced by Apple in 2014, Swift is the language of choice for iOS development. It is very simple to use, with Apple even promoting its use to children creating their own apps, and is known for how it supports a prospective Apple developer, it is one of the best languages to pick up, boasting both speed and simplicity.


  • Designed to be simple to use
  • Promotes both Object Oriented and Functional programming
  • Large support in the Apple community
  • Easy to integrate with existing Objective-C code
  • Safer code with less errors
  • Fast and concise

Cons:
  • Newer versions are incompatible with older versions of Swift
  • Only popular within Apple products
  • Incompatible with languages such as C++

Go

Go is designed by Google for systems programming, and fulfills its task while maintaining ease of use unseen in its predecessor – C. The language's effectiveness in Systems programming is not to be overlooked, nor is the support of the renown technological giant's support. A reliably simple and efficient language, Go may be the perfect next step in a programmer's journey, especially with its similar syntax to C.


  • Very easy to use and pick up
  • Object Oriented Programming
  • Strong support from Google
  • Great at large systems programming
  • Concise code

Cons:
  • Less popularity and general support than other languages
  • Less versatility than other languages (like Swift)
  • Less efficient(larger file sizes)
